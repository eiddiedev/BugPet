# BugPet

BugPet is a lightweight desktop pet MVP built with Tauri, React, and TypeScript.

## Step 1

This repository is currently aligned to the Step 1 baseline:

- Tauri + React + TypeScript project structure is in place
- Vite frontend builds successfully
- Tauri config lives in `src-tauri/`
- Electron leftovers were removed so the startup path is unambiguous

## Commands

- `npm install`
- `npm run dev`
- `npm run tauri:dev`
- `npm run build`
